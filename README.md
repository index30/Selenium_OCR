# Selenium OCR

## Environments

- macOS >= 10.14.5
- Python >= 3.5
    - chromedriver-binary == 75.0.3770.8.0
    - selenium == 3.141.0
    - pyocr == 0.7
- [Tesseract](https://github.com/tesseract-ocr/tesseract/wiki)

## How to Work
```
$ pip3 install -r requirements.txt
$ python3 main.py
$(option) python3 main.py (URL)
```
